# Governance

Turnkey Site is maintainer-led. Maintainers set the public information
architecture, approve product and release claims, review licensing and safety
boundaries, and authorize promotion to `main`.

Routine corrections may be accepted after one maintainer review. Changes to
repository ownership, public release claims, security guidance, licensing,
course architecture, or deployment policy require explicit owner approval.
Disagreements should be resolved with current product behavior, reproducible
tests, and reader outcomes as primary evidence.

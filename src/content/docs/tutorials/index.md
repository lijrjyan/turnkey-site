Tutorials are for doing. Each one states prerequisites, runs a complete task,
shows the expected evidence, and points to the first debugging surface.

- [Inspect run artifacts](./inspect-artifacts/): connect one case to its events,
  metric contribution, and resolved run.
- [Calibrate a threshold](./calibrate-threshold/): produce a calibration
  artifact without confusing calibration data with final evaluation data.
- [Learn Turnkey](../learn/): use the ordered course when you need the mental
  model before the task.

More tutorials will be added only when their runtime path is executable and
versioned. Reference-only stubs are kept out of this lane.
